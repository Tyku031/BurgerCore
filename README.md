# BurgerCore

## Introduction

BurgerCore is a core plugin made by bundling and improving my older plugins that were made for the Minecraft server Burgercraft during the Covid-19 pandemic (2020-2021). I also added features that didn't exist in those older plugins. These older plugins of mine include Moderation&Essentials, Permissions and Hub.

## Requirements

This plugin is made for Minecraft version 1.20.4, although it will likely work on some newer/older versions. None of my other plugins are required for this plugin.
